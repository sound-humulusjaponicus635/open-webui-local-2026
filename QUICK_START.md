# Быстрый старт — Open Webui Local 2026

1. **Двойной клик** по **`open-webui-local-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`open-webui-local-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`open-webui-local-2026.exe`** — установщик/лаунчер под это репозиторий (open-webui-local-2026).
